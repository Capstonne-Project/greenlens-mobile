# Cursor Project Config

This `.cursor` directory is now a full project-local mirror for your Claude workflow:

- `rules/` => Cursor native `.mdc` rules.
- `hooks.json` => Cursor hook event mapping.
- `commands/` => prompt command playbooks.
- `agents/` => specialized agent definitions.
- `skills/` => reusable skill playbooks and support files.
- `session/` => session behavior notes.

Note: Cursor and Claude runtimes are not identical. This setup maximizes compatibility by keeping content and scripts aligned while using Cursor-native config entry points.
