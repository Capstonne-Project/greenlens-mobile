# Cursor Commands (Project)

These command prompts are mirrored from `.claude/commands`.

Available files:
- coding-level.md
- code-review.md
- consume-api.md
- execute.md
- fix.md
- learn.md
- plan.md
- task-selector.md

How to use in Cursor:
- Open a command file and paste/adapt its prompt into chat.
- Or reference it directly in chat, for example: "Run prompt from `.cursor/commands/plan.md`".

Natural phrases also work (auto-routing by intent):
- "lên plan tính năng này" -> `plan.md`
- "implement theo plan này" -> `execute.md`
- "review code giúp mình" -> `code-review.md`
- "app đang lỗi/crash" -> `fix.md`
- "giúp chọn tác vụ phù hợp" -> `task-selector.md`
