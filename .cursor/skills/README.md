# Cursor Skills (Project)

This folder mirrors `.claude/skills` so your existing skill playbooks are available in Cursor project context.

Included skill packs:
- all
- build-app
- component
- debug
- feature
- frontend-design
- performance
- screen
- security
- skill-creator

Usage:
- Ask Cursor to follow a skill file, e.g. `.cursor/skills/debug/SKILL.md`.
- For composite skills (like `skill-creator`), supporting scripts/assets are copied too.
